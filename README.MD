Système d'Aide à la Décision Crédit - Coopération 
Objectif : Transition du scoring comptable classique vers un modèle d'apprentissage statistique pour réduire les impayés et optimiser le temps de traitement des dossiers.

Méthodologie :

Ingénierie de données : Calcul de ratios financiers (Solvabilité, Endettement).

Modélisation : Régression Logistique avec validation par AUC.

Performance : Modèle capable de distinguer les risques avec une précision de 90%.

Usage : Aide au comité de crédit pour le filtrage automatique des dossiers à haut risque.
